Slijedite sljedeæe upute:
1. Instalirati Virtual Serial Port Emulator
2. Pokrenuti Virtual serial port Emulator
3. Device -> Create
   Device Type: Pair -> Next
   Virtual serial Port 1: COM3
   Virtual serial Port 2: COM4
   Emulate baud rate -> Chack
   Finish
   (Ne zatvarati ovaj program)
4. U ISIS Proteusa otvoriti "Elektricna_prog.DSN"
   Klik  na PLAY
5. Otvoriti Matlab i anvesti putanju do foldera "Simulacija sistema"
6. Pokrenuti fajl "incijalizacija.m"
7. Pokrenuti "UpravljanjeKoracnimMotorom.m"
8. klik na Test
9. Prije zatvranja Matlaba pokrenuti "deincijalizacija.m"
